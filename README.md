## Описание проекта

Проект состоит из трех приложений для анализа временных рядов:
1. Анализ курса рубля к USD и EUR
2. Анализ преступности в России
3. Анализ численности населения России

Каждое приложение позволяет загружать данные из файла, просматривать их в таблице, строить графики и делать прогноз методом скользящей средней.

---

## Задание 1: Анализ курса рубля

**Файл:** `ruble.py‎`

**Что делает программа:**
- Загружает данные из Excel-файла `currency_data.xlsx` с колонками Date, RUB_USD, RUB_EUR
- Показывает таблицу с датами и курсами валют (USD и EUR)
- При изменении курса более чем на 1 рубль ячейка окрашивается: красный - рост, зеленый - падение
- Строит график курсов за выбранный период (все данные, 7, 14, 30, 90 дней)
- Рассчитывает максимальный рост и падение курса за один день с указанием дат
- Делает прогноз на N дней методом скользящей средней
- Прогноз отображается на графике пунктирной линией и серой зоной

## Задание 2: Анализ преступности

**Файл:** `crime.py‎`

**Что делает программа:**
- Загружает данные из Excel-файла `crime_data.xlsx` за 15 лет
- Автоматически определяет названия колонок (поддерживает русские и английские варианты)
- Отображает таблицу с 6 видами преступлений: убийства, разбои, кражи, мошенничество, наркотики, экономические
- Строит графики всех видов преступлений разными цветами
- Определяет, какой вид преступности снизился больше всего за 15 лет, а какой - меньше всего
- Делает прогноз на N лет для каждого вида преступлений (пунктирные линии на графике)
- Поддерживает экспорт графика в PNG, PDF, SVG

**Архитектура:** Программа построена по принципам SOLID:
- Интерфейсы для загрузки данных, прогнозирования и анализа
- Легко добавить новый тип преступности или источник данных

## Задание 3: Анализ численности населения

**Файл:** `population.js`

**Что делает программа:**
- Загружает JSON-файл с данными о населении (год и численность в млн)
- Отображает таблицу с расчетом годового изменения в процентах
- Зеленым цветом показывает рост, красным - убыль
- Рассчитывает максимальный прирост и максимальную убыль населения за год
- Строит график численности населения
- Делает прогноз на N лет методом скользящей средней
- Поддерживает экспорт графика в PNG
